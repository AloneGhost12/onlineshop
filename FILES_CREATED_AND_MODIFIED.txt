NEW FILES CREATED (4)
=====================

1. /src/context/DebugContext.js (5.3 KB)
   Purpose: Global debug state management using React Context
   Features:
   - Captures errors, logs, API calls, performance metrics, WebGL status
   - Auto-intercepts console methods (log, error, warn, info)
   - Catches global errors and unhandled rejections
   - Detects WebGL support and capabilities
   - Provides useDebug() hook for accessing debug data
   - Methods: addError, addLog, addApiCall, updatePerformanceMetrics, clearLogs
   - Memory management: Keeps last 500 errors, 1000 logs, 500 API calls

2. /src/hooks/usePerformanceMonitor.js (1.2 KB)
   Purpose: Real-time performance monitoring hook
   Features:
   - Measures FPS (frames per second) using requestAnimationFrame
   - Tracks JavaScript heap memory usage
   - Updates metrics every 1000ms
   - Non-blocking implementation
   - Automatically integrates with DebugContext

3. /src/lib/debugUtils.js (7.4 KB)
   Purpose: Debug utility functions and helpers
   Functions:
   - setupNetworkMonitoring() - Intercepts fetch requests
   - getPerformanceMetrics() - Collects navigation and resource timing
   - formatBytes() - Converts bytes to readable format
   - parseErrorStack() - Extracts function/file info from stack traces
   - exportLogsAsJSON() - Downloads logs as JSON file
   - exportLogsAsCSV() - Downloads logs as CSV file
   - getWebGLInfo() - Retrieves WebGL capabilities
   - getErrorStatistics() - Calculates error metrics
   - getApiStatistics() - Calculates API call metrics
   - filterLogsByLevel() - Filters logs by severity
   - filterLogsByMessage() - Searches logs by text

4. /src/components/admin/DebugDashboard.jsx (24 KB)
   Purpose: Main debug dashboard UI component
   Features:
   - 5 tabs: Overview, Errors, Logs, API, WebGL
   - Admin-only access control
   - Real-time statistics with 8 metric cards
   - Error tracking with stack traces
   - Console log viewer with search/filter
   - API call monitoring with response preview
   - WebGL status and capability reporting
   - Export to JSON/CSV functionality
   - Clear logs, search, and copy-to-clipboard
